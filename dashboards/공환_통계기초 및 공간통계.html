통계학 기초 상식

공간 분석을 시작하기 전, 반드시 알아야 할 데이터의 분포와 가설 검정의 기초 개념입니다.

분산 (Variance)

데이터가 평균으로부터 얼마나 퍼져 있는지를 나타내는 지표. 편차의 제곱의 평균입니다.

σ² = Σ(X - μ)² / N

표준편차 (Standard Deviation)

분산의 양의 제곱근. 데이터가 평균을 중심으로 얼마나 퍼져 있는지를 원래 단위로 보여줍니다.

σ = √Variance

Z-스코어 (Z-Score)

어떤 데이터가 평균으로부터 표준편차의 몇 배만큼 떨어져 있는지를 나타내는 표준화 지수입니다.

Z = (X - μ) / σ

가설 검정의 기초

⚖️ T-검정 (T-Test)

두 집단 간의 평균 차이가 통계적으로 유의미한지 검증할 때 사용합니다. (예: 서울과 부산의 평균 집값 차이 비교)

📈 F-검정 (F-Test)

두 집단 간의 분산 차이가 유의미한지, 혹은 다중회귀분석에서 모델 전체의 설명력이 유의미한지 검증할 때 사용합니다.

정규분포 (Normal Distribution)

자연계와 사회 현상에서 가장 흔하게 나타나는 종 모양의 데이터 분포입니다. 통계적 추론의 근간이 됩니다.

QGIS/GeoDa 수치 정규화 (Normalization)

❓ 왜 정규화를 해야 할까?

  • 단위 통일: 면적(m²), 인구(명), 금액(원) 등 서로 다른 단위의 변수들을 동일한 척도로 비교하기 위해.
  • 면적의 영향 제거: 공간 단위(행정동 등)의 크기가 다를 경우, 단순 총량(예: 총 범죄 건수)을 비교하면 면적이 큰 곳이 무조건 높게 나타나는 오류가 발생합니다. 인구당, 면적당 비율로 변환해야 합니다.
  • 모델 안정성: 회귀분석 시 변수 값의 스케일 차이가 크면 계산 과정에서 왜곡이 발생할 수 있습니다.

🛠️ GeoDa에서의 사용 방법

  1. 데이터 테이블 열기 (Table).
  2. 새로운 변수 추가 (Add Variable).
  3. Calculator(계산기) 실행.
  4. 스탠다드화(Standardize): (변수 - 평균) / 표준편차 수식을 적용하여 Z-score 형태로 변환. (GeoDa 내장 함수 이용 가능)
  5. 비율 계산: Rate 탭을 이용하여 이벤트 수(Event) / 기준 인구(Base) 형태의 비율 변수를 직접 생성할 수 있습니다.

공간통계학 입문

데이터가 가진 '위치' 정보가 미치는 영향을 수학적 모델로 풀어내는 방법론을 학습합니다.

📍 공간통계학이란?

"모든 것은 다른 모든 것과 관련되어 있지만, 가까운 것은 먼 것보다 더 관련되어 있다." (Tobler의 지리학 제1법칙).

일반 통계학과 달리, 데이터의 지리적 위치와 이웃 관계(공간적 자기상관성)를 분석에 직접 포함시키는 학문입니다.

변수의 이해

종속변수 (Dependent Variable, Y)

우리가 원인을 알고 싶어 하거나 예측하고자 하는 결과값 (예: 집값, 범죄율)

독립변수 (Independent Variable, X)

종속변수에 영향을 미칠 것으로 예상되는 원인값들 (예: 지하철역 거리, 공원 면적)

공간 가중치 (Spatial Weights)

어느 지역이 서로 '이웃'인지 정의하는 행렬(W) 설정 방법

  • 인접성 (Contiguity): 경계선이 닿아있으면 이웃. (Rook: 변 공유, Queen: 점/변 공유)
  • 거리 (Distance): 중심점 간의 거리가 특정 반경(Band) 내에 있으면 이웃, 혹은 거리에 반비례하여 가중치 부여.
  • K-최근린 (K-Nearest): 거리에 상관없이 가장 가까운 K개의 지역을 이웃으로 설정.

회귀분석 모델 발전 과정

회귀분석은 독립변수(들)가 종속변수에 미치는 영향을 선형 방정식으로 추정하는 기법입니다. 공간 효과를 어떻게 처리하느냐에 따라 모델이 진화합니다.

1. 다중회귀분석 & OLS (최소제곱법)

식: Y = β₀ + β₁X₁ + β₂X₂ + ... + ε

여러 개의 독립변수를 사용하여 종속변수를 설명하는 가장 기본적인 선형 모델. '공간적 영향'은 전혀 고려하지 않습니다. 잔차(오차)의 제곱합을 최소화하는 방식으로 계수를 찾습니다.

2. 공간 자기회귀 모델 (SLM / SEM)

OLS의 잔차에 공간적 패턴이 남아있을 때(공간 자기상관성 존재 시) 이를 모델에 포함시킵니다.

SLM (공간시차모델)
식: Y = ρWY + Xβ + ε
"내 결과(집값)는 이웃의 결과(이웃 집값)에 직접적인 영향을 받는다."
SEM (공간오차모델)
식: Y = Xβ + u, u = λWu + ε
"모델에 포함되지 않은 숨겨진 요인(오차)들이 공간적으로 뭉쳐서 영향을 미친다."

3. 지리적 가중회귀분석 (GWR)

식: Yᵢ = β₀(uᵢ, vᵢ) + Σβₖ(uᵢ, vᵢ)Xᵢₖ + εᵢ

전역적(Global) 모델인 앞선 모델들과 달리, 지역적(Local) 모델입니다. 변수 간의 관계가 공간에 따라 일정하지 않다는 '공간적 이질성(Spatial Heterogeneity)'을 가정합니다. 각 지역마다 다른 회귀 계수(β)가 산출됩니다.

4. 기타 고급 모델

SDM (공간더빈모델): 내 결과가 이웃의 종속변수뿐만 아니라 이웃의 '독립변수'에도 영향을 받음을 가정한 복합 모델.
패널 공간회귀 (Panel Spatial Regression): 시간적 변화와 공간적 상호작용을 동시에 고려하는 시공간 분석 모델.

모델 검증 및 실무 가이드

분석을 돌리는 것보다 중요한 것은 결과를 해석하고 오류를 찾아 해결하는 과정입니다.

🚨 분석 시 발생하는 주요 문제 및 해결법

1. 다중공선성 (Multicollinearity)

문제: 독립변수들끼리 너무 높은 상관관계를 가지는 현상. (예: '방의 개수'와 '집의 면적'을 둘 다 독립변수로 넣을 경우). 모델의 계수가 불안정해지고 해석이 불가능해집니다.

분석 방법: VIF (Variance Inflation Factor, 분산팽창지수) 확인.
판단 기준: VIF > 10 이면 다중공선성 의심.
해결법: VIF가 높은 변수 중 논리적 덜 중요한 변수를 제거하거나, PCA(주성분분석)를 통해 변수를 축소합니다.

2. 잔차 분석 (Residual Analysis)

문제: 잔차(실제값 - 예측값)는 무작위로 분포해야 합니다. 특정 패턴(공간적 뭉침)이 남아있다면 OLS 모델이 공간 효과를 잡아내지 못했다는 뜻입니다.

분석 방법: OLS 수행 후 산출된 잔차(Residual) 변수에 대해 Moran's I (모란지수) 분석 실행.
판단 기준: Moran's I p-value가 0.05 미만이면 잔차의 공간 자기상관성 존재.
해결법: 단순 OLS를 버리고 SLM, SEM, GWR 등의 공간 회귀 모델로 전환해야 합니다.

🔍 OLS -> 모델 선택 과정 (LM TEST)

어떤 공간 모델(SLM vs SEM)을 써야 할지는 마음대로 정하는 것이 아니라 통계적 검정(LM Test)을 거쳐 결정합니다.

1. 먼저 OLS 분석을 실행하고 LM Test(Lagrange Multiplier Test) 결과를 확인합니다.

2. LM-Lag (SLM 테스트)와 LM-Error (SEM 테스트)의 유의확률(p-value)을 확인합니다.

  • 둘 다 안 유의함 -> 그냥 OLS 쓰면 됨.
  • Lag만 유의함 -> SLM 선택
  • Error만 유의함 -> SEM 선택
  • 둘 다 유의함 -> Robust LM-Lag와 Robust LM-Error 결과를 비교하여, 값이 더 크고 유의확률이 더 낮은(더 통계적으로 강력한) 모델을 최종 선택합니다.
