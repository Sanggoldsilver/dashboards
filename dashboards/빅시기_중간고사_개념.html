```html 문과생을 위한 데이터 대시보드 (4~6주차)

문과생 데이터 번역기

4주차 ~ 6주차 통합 강의노트

  • 4주차: 데이터 요약
  • 5주차: 시각화 기초
  • 6주차: 고급 시각화

4주차. 방대한 이야기 한 줄로 요약하기

1,000페이지짜리 두꺼운 소설을 뒷면의 '줄거리 요약본'으로 압축하는 마법, 기술통계를 배웁니다.

수십만 개의 도서관 대출 데이터(엑셀 표)를 눈으로 다 읽을 수는 없습니다. 그래서 우리는 컴퓨터에게 "이 데이터들의 전반적인 분위기가 어때? 한 줄로 요약해 봐!"라고 명령해야 합니다. 이 요약본을 데이터 세계에서는 기술통계(Descriptive Statistics)라고 부릅니다.

평균(Mean)과 중앙값(Median)

평균은 모든 값을 더해서 공평하게 나눈 값입니다. 하지만 치명적인 단점이 있죠. 평범한 동네 식당에 갑자기 '이재용 회장'이 들어오면, 그 식당 손님들의 '평균 재산'이 수백억으로 왜곡됩니다.

그래서 중앙값이 필요합니다. 손님들을 재산 순서대로 1등부터 꼴등까지 한 줄로 세웠을 때, 정확히 한가운데 서 있는 사람의 재산을 보는 것이죠. 이게 현실을 더 정확히 반영할 때가 많습니다.

분산과 표준편차

이름만 들어도 머리가 아프죠? 소설에 비유해 봅시다. 이야기의 전개가 얼마나 극적으로 널뛰기를 하는지를 나타내는 '막장 드라마 지수'라고 생각하세요.

데이터들이 평균(평범한 일상)에 오순도순 모여있으면 표준편차가 작고, 평균에서 벗어나 극단적으로 미쳐 날뛰면(막장 전개) 표준편차가 커집니다.

궁극의 요약 마법: describe()

평균 구하고, 중앙값 구하고, 일일이 명령하기 귀찮으신가요? 엑셀 표(데이터프레임) 뒤에 .describe()라는 주문 하나만 붙이면, 데이터의 개수, 평균, 최소값, 최대값, 중앙값까지 책의 요약본처럼 한 방에 리포트로 출력해 줍니다.

# df(데이터야) .describe(너 자신을 요약해서 설명해봐) df.describe()

5주차. 숫자를 그림으로 번역하기

답답한 엑셀 표의 숫자들을 아름다운 그래프(인포그래픽)로 그려내는 붓과 물감을 준비합니다.

파이썬 세계에는 밥 아저씨처럼 "참 쉽죠?" 하면서 그림을 그려주는 전속 화가가 살고 있습니다. 이 화가의 이름이 바로 맷플롯립(Matplotlib)입니다. 우리는 이 화가에게 어떤 그림을 어떻게 그릴지 지시만 내리면 됩니다.

선 그래프 (plot)

선 그래프는 소설의 '서사 구조(발단-전개-절정-결말)'를 보여줄 때 씁니다. 시간이 흐름에 따라 주식 가격이 어떻게 변했는지, 연도별 도서 대출량이 어떻게 변했는지 등 '흐름과 추세'를 보여줄 때 가장 완벽한 그림입니다.

# plt(화가야) .plot(선 그래프로 그려줘) plt.plot(시간, 대출량)

막대 그래프 (bar)

막대 그래프는 캐릭터들의 '능력치 비교'를 할 때 씁니다. 로맨스 소설, 추리 소설, SF 소설 중 어떤 장르가 가장 많이 대출되었는지 나란히 세워두고 키를 재보는 겁니다. '누가 누가 더 크나 비교'할 때는 막대가 최고죠.

# plt(화가야) .bar(막대 그래프로 그려줘) plt.bar(소설장르, 대출량)

그림을 다 그렸으면 보여줘야지: show()

컴퓨터 메모리 속에서 화가가 열심히 그림을 완성했습니다. 하지만 짠! 하고 모니터 화면에 전시하려면 마지막에 꼭 "이제 보여줘!"라는 명령을 내려야 합니다.

# plt(화가야) .show(지금까지 그린 그림을 화면에 띄워!) plt.show()

6주차. 나만의 데이터 전시회 열기

단순한 스케치를 넘어, 커다란 도화지를 반으로 나누고 여러 액자를 거는 고급 전시 기법을 배웁니다.

5주차에서는 화가(plt)에게 무작정 "그려!"라고 명령했다면, 6주차 객체지향 API 방식에서는 우리가 직접 갤러리 관장이 됩니다. 도화지(Figure)를 직접 사고, 그 위에 여러 개의 액자(Axes)를 우리 마음대로 배치하는 세밀한 작업입니다.

도화지와 액자: subplots()

"큰 벽면(Figure) 하나를 준비하고, 거기에 액자(Axes) 두 개를 가로로 나란히 걸어줘!" 라고 명령하는 방법입니다. 이제 액자 1번에는 선 그래프를, 액자 2번에는 막대 그래프를 각각 따로따로 그릴 수 있게 됩니다.

# 큰 도화지(fig)와 액자들(axs)을 한 번에 세팅해줘 fig, axs = plt.subplots(1, 2)

산점도 (scatter)

밤하늘에 별을 찍듯 점을 콕콕 찍는 그래프입니다. 소설의 '인물 관계도'와 비슷하죠. '책의 페이지 수'와 '대출 횟수' 사이에 어떤 은밀한 관계(두꺼운 책일수록 안 빌려갈까?)가 있는지 알아볼 때 점들이 찍힌 모양을 보며 추리해 냅니다.

히스토그램 (hist)

막대그래프와 닮았지만 다릅니다. 히스토그램은 '도수분포표'를 시각화한 것입니다. 대출 건수가 '0~10번인 책은 몇 권, 11~20번인 책은 몇 권' 이런 식으로 구간을 나누어 데이터가 어디에 뚱뚱하게 몰려있는지(분포)를 보여주는 엑스레이 같은 그래프입니다.

축하합니다! 이제 당신은 데이터를 요약하고, 아름다운 그림으로 번역할 수 있는 통역가가 되었습니다.
```
